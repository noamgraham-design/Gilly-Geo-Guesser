הצהרת נגישות

אנו מחויבים להנגשת האתר לכלל המשתמשים, לרבות אנשים עם מוגבלויות.

רמת הנגישות

WCAG 2.0 רמה AA

האתר פועל לעמידה בדרישות תקן ישראלי ת"י 5568 (המבוסס על WCAG 2.0 רמה AA), בהתאם לחוק שוויון זכויות לאנשים עם מוגבלות, תשנ"ח-1998, ותקנות שוויון זכויות לאנשים עם מוגבלות (התאמות נגישות לשירות), תשע"ג-2013.

תאריך עדכון הצהרה זו: מרץ 2026.

אמצעי הנגישות שיושמו

  • תמיכה בהגדלת טקסט עד 200% ללא פגיעה בתוכן (הסרת חסימת זום)
  • אינדיקטורי פוקוס גלויים לכל האלמנטים האינטראקטיביים
  • תגיות ARIA לדיאלוגים, הודעות סטטוס ואזורים דינמיים
  • ניהול פוקוס בחלונות קופצים (focus trap)
  • קישור דילוג לתוכן הראשי
  • עדכון שפת המסמך (lang) בעת מעבר בין עברית לאנגלית
  • תמיכה בהגדרת "הפחת תנועה" של מערכת ההפעלה
  • תוויות נגישות (aria-label) לכפתורים ללא טקסט גלוי

מגבלות ידועות

  • מפה אינטראקטיבית (Leaflet.js): המפה היא הממשק המרכזי של המשחק. ניווט מלא בקורא מסך מוגבל מאחר שמפות אינטראקטיביות הן בגדר "אפליקציית ריץ'" (role="application"). המשחק מוגדר כ-בידור ולא כשירות חיוני; אנו ממשיכים לבחון שיפורים.
  • תמונות בטבלת תוצאות: חלק מהאייקונים הם אמוג'י קלים ואינם נושאים משמעות פונקציונלית.

Accessibility Statement

We are committed to making this site accessible to everyone, including people with disabilities.

Compliance Level

WCAG 2.0 Level AA

This site strives to conform to Israeli Standard IS 5568, based on WCAG 2.0 Level AA, as required under Israel's Equal Rights for Persons with Disabilities Law (1998) and related accessibility regulations.

This statement was last updated: March 2026.

Accessibility Measures Implemented

  • Text can be resized up to 200% without loss of content (zoom blocking removed)
  • Visible focus indicators on all interactive elements
  • ARIA roles and labels on dialogs, status messages, and dynamic regions
  • Focus trap management in modal dialogs
  • Skip-to-main-content link
  • Document language (lang attribute) updated on Hebrew/English toggle
  • Respects OS "Reduce Motion" preference — animations are suppressed
  • Accessible labels (aria-label) on icon-only buttons

Known Limitations

  • Interactive map (Leaflet.js): The map is the primary game interface. Full screen-reader navigation is limited because interactive maps are inherently visual (exposed as role="application"). The site is entertainment, not an essential service; we continue to explore improvements.
  • Scoreboard icons: Some icons are decorative emoji and carry no functional meaning.

Technologies Used

  • HTML5, CSS3, JavaScript (Vanilla)
  • Leaflet.js — interactive mapping
  • Firebase Authentication & Firestore
  • Google Fonts (Alef, Playfair Display, Source Serif 4)

Testing Methods

  • Automated testing with Lighthouse (Chrome DevTools) — accessibility score 90+
  • Full keyboard-only navigation — tab order and focus cycle verified in all modals
  • Manual testing with VoiceOver (macOS) and NVDA (Windows)
  • Zoom testing to 200% in Chrome, Firefox, and Safari
  • OS "Reduce Motion" setting tested
